Perprof-py

A Python Package for Profiling Optimization Software

In the Mathematical Programming field, we are frequently required to compare two solvers. An usual way to do so, is the performance profile [1]. This tools implements the performance profile, as well as the data profile [2] and the extended performance profile [3].

For more information, check the documentation, the GitHub repository, or ping us at Gitter.

Installation

  • Install Python 3. The python and pip commands below are for Python 3. Your system may them python3 and pip3.
  • Download perprof-py v1.1.2 ( zip , tar.gz )
  • Uncompress the file and enter the folder perprof-py-1.1.2
  • Enter the commands 
    
          sudo pip install -r REQUIREMENTS
          sudo python setup.py install
  • (Optional) Test with sudo python setup.py test
